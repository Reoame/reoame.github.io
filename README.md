## 关于我

我是 Reoame，一名学生，平常喜欢打羽毛球，有空会写一些开源项目.在我小的时候，我开始接触编程，当时呢我大概只有10岁左右，当时英语不是很好，所以我基本不想看GitHub上的代码，所以我基本都是看csdn上的教程。

后来我学会了Python，开始接触到开源项目，我开始关注GitHub上的大佬们，我开始学习前端，我开始学习机器学习，我开始学习深度学习，我开始学习计算机视觉，我开始学习人工智能。

后来我想着去做一名红客，我开始学习前端，网络安全，渗透测试，开始向Python高级编程进击。我时常会找一些Github上的项目来进行学习。


## 关于博客建设
2024年1月1，我开始想这自己去部署一个博客，但是由于种种原因，一直没有动手。直到最近，我发现了GitHub Pages，于是我开始了这个博客的建设之旅。

我开始在网上学习前端，学习写脚本（JavaScript），后来我知道有typecho hexo wordpress这些博客系统，于是我选择了hexo。

我会在这里记录一些学习笔记，分享一些心得体会，也会分享一些我认为有趣的东西。

但是基本没人访问过，后来我不再更新博客，也不再维护，所以博客的文章也就慢慢淡出视线。
