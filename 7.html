我的笔记软件Qnote

2025年2月22日 · 编程

Qnote是一款基于 Python 编写的跨平台的 Markdown 笔记应用，使用 Python 构建，可以快速部署到Windows系统上。

安装

Qnote 目前支持 Windows 系统，你可以从 GitHub Releases 页面下载安装包。

使用

Qnote 使用 Markdown 语法来编写笔记，你可以在编辑器中直接编写笔记内容，也可以使用快捷键快速插入常用语法。

使用规范🤗

Qnote 是一个 Markdown 笔记应用，它的使用规范和 Markdown 语法一样，你可以参考 Markdown 语法指南 来编写笔记。

在使用这个软件过程中不允许出现任何侵犯他人权益的内容，包括但不限于色情、暴力、赌博、政治、反动、诽谤等违反法律的内容。

不能用于商业用途，请不要将 Qnote 用于任何违法用途。

结语

Qnote 是一个开源的跨平台的 Markdown 笔记应用，你可以在 GitHub Releases 页面下载安装包。

Qnote 还处于开发阶段，它的功能还不完善，欢迎你在 GitHub 上提出你的宝贵意见。
